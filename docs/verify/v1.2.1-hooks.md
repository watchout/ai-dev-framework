## 2. 境界値テスト [必須]

| 項目 | 境界 | 期待値 |
|---|---|---|
| hook 実行時間 | 5s 直前 | warn log なし、exit 0 |
| hook 実行時間 | 5s 超過 | warn log あり、exit code は script 依存 |
| settings.json 既存 PreToolUse | 0 件 | merge で 1 件追加 |
| settings.json 既存 PreToolUse | 1 件 | merge で 2 件 (append) |
| `framework hook test --hook=Stop` | hook 不在 | exit 2 + reason |
| hook-log retention | 30 日丁度 | 保持される |
| hook-log retention | 31 日超 | rotate で削除 |

## 3. 異常系テスト [必須]

| 入力 | 期待するエラー | エラーコード |
|---|---|---|
| `bash gh pr merge 5 --admin` (PreToolUse 経由) | "governance violation: admin merge" | exit 2 |
| `bash git push origin main` (PreToolUse 経由) | "governance violation: direct push to main" | exit 2 |
| `bash rm -rf /` | "dangerous command pattern" | exit 2 |
| Stop hook で test fail | `{"decision":"block","reason":"Test suite must pass before completion"}` | exit 0 (JSON で block) |
| settings.json schema 違反 (matcher 欠落) | `validate` で stderr "matcher missing at PreToolUse[0]" | exit 2 |
| `framework hook init` 連続起動 (lock 衝突) | "Another framework hook init in progress" | exit 1 |
| `framework hook init` で既存 settings 衝突、`--force` なし | "Conflict at hooks.PreToolUse[0]; use --force" | exit 2 |

| 攻撃ベクタ | 想定結果 |
|---|---|
