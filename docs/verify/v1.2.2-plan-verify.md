## 3. 異常系テスト [必須]

[推測: IMPL §4.1 例外分類 + SPEC §7 受入基準から導出]:

| 入力 | 期待するエラー | エラーコード |
|---|---|---|
| `framework dispatch task-1 bot --mode=implement` で Plan 不在 | `PlanRequiredError` | exit 2 |
| Plan mtime 25h 前 | `PlanStaleError` | exit 2 |
| post-merge verify で test fail | `VerifyFailureError` | exit 2、verified=false 永続化 |

| `ADF_PLAN_BYPASS=1` で implement | warn + audit log + Discord、実行は継続 | exit 0 |
