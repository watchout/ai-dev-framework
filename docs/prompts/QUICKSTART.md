# フレームワーク適用クイックスタート

> CLI不要。Claude Code セッションにプロンプトを貼るだけ。

---

## 新規プロジェクト

1. プロジェクトのディレクトリで Claude Code セッションを開く
2. **[new-project.txt](new-project.txt)** の内容を**全文コピー**して貼り付ける
3. あとは対話形式で進む

GitHub で直接見る場合:
https://github.com/watchout/ai-dev-framework/blob/main/docs/prompts/new-project.txt

---

## 既存プロジェクト

1. プロジェクトのディレクトリで Claude Code セッションを開く
2. **[existing-project.txt](existing-project.txt)** の内容を**全文コピー**して貼り付ける
3. 現状スキャン → SSOT逆生成 → ギャップ対応

GitHub で直接見る場合:
https://github.com/watchout/ai-dev-framework/blob/main/docs/prompts/existing-project.txt
