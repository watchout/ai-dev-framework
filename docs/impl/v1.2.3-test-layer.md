}

## 3. シーケンス [推測 init]

### 3.1 E2E wake test flow (Mermaid)
```mermaid
sequenceDiagram
  participant Test as e2e test runner
  participant DB as message_queue
  participant SD as state-daemon
  participant Tmux as tmux session
  Test->>DB: INSERT pending row
  DB->>SD: pg_notify
  SD->>Tmux: tmux send-keys 'check inbox\n'
  Test->>Tmux: capture-pane verify
  Test->>DB: SELECT last_wake_attempt_at verify
```
